# 🛒 移动端购物商城（LuckySHOP_app）

一个基于纯前端技术开发的移动端购物商城网站项目，页面结构规范，功能完整，适合作为前端学习与实战练习项目。

---

## 📱 项目简介

本项目是一个标准结构的移动端购物商城网站，包含完整的电商基础功能模块，实现了用户登录、商品浏览、购物车管理、订单结算以及个人信息修改等功能。

项目基于前端技术栈开发，并通过 **AJAX** 技术实现异步数据交互，提高用户体验。

---

## 🏗️ 项目结构

项目主要包含四个核心页面模块：

- 首页（Home）
- 分类页（Category）
- 购物车页（Cart）
- 个人中心页（Profile）

页面结构清晰，符合移动端商城常见布局规范。

## 🚀 功能实现

### 👤 用户功能
- 用户登录功能
- 登录状态校验
- 修改个人信息功能

### 🛍️ 商品功能
- 商品列表展示
- 分类筛选

### 🛒 购物车功能
